Laboratorio #4

Preguntas

1. ¿Qué diferencias y semejanzas hay entre Java y JavaScript?

Ambos son lenguajes orientados a objetos y tienen una sintaxis similar. Sin embargo, Java es un lenguaje compilado y estático, mientras que Javascript es interpretado y dinámico.

2. ¿Qué métodos tiene el objeto Date?

  • getTime(): Devuelve el tiempo en milisegundos desde el 1 de enero de 1970.
  • getMinutes(): Devuelve los minutos de la hora.
  • getHours(): Devuelve las horas del día.
  • setTime(): Establece el tiempo en milisegundos.
  • setFullYear(): Establece el año.

    • 3. ¿Qué métodos tienen los arreglos?

  • push(): Añade un elemento al final del arreglo.
  • pop(): Elimina el último elemento del arreglo.
  • shift(): Elimina el primer elemento del arreglo.
  • splice(): Añade o elimina elementos de un arreglo.
  • length: Devuelve la longitud del arreglo.

    • 4. ¿Cómo se declara una variable con alcance local dentro de una función?

    Mediante la palabra reservada 'let', dentro de la misma función.

    5. ¿Qué implicaciones tiene utilizar variables globales dentro de funciones?

    Puede derivar en errores y fallos de seguridad, ya que podrían ser modificadas por cualquier función.
